
1. To install yarn package manager ==> npm install -g yarn
2. To check yarn version that is installed ==> yarn --version
3. To install 3rd party libraries ==> yarn add react-router-dom axios moment react-moment react-tostify react-icons
4. To install react bootstrap ==> npm install react-bootstrap bootstrap
5. To use bootstrap import "bootstrap/dist/css/bootstrap.min.css" in "global.css".
5. To install json-server ==> yarn add global json-server or npm install -g  json-server
6. To start json-server ==> json-server --watch backend/db.json --port=5000
7. To run the project ==> yarn start or npm start
